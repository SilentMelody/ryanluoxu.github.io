Debug 断点调试

Eclipse 快捷键

  • 内容提示： alt+/
  • 快速修复： ctrl+1
  • 导包： ctrl+shift+o
  • 格式化代码块： ctrl+shift+f
  • 向前向后： alt+方向键
  • 查看方法说明： f2
  • 重置透视图： Windows - Reset Perspective
  • 更改大写： ctrl+shift+x
  • 更改小写： ctrl+shift+y
  • 复制一行： ctrl+alt+上下键
  • 移动一行： alt+上下键
  • 查看继承关系： ctrl+t
  • 查看源码： ctrl+shift+t
  • 查看快捷键： ctrl+shift+L

相当一个类型的变量。
在编写集合时，限制了集合的处理类型，从而把运行问题，转为编译问题。

注意的问题：

  • 泛型的对象不能是基本类型。
  • new时，前后泛型要一致。
  • 泛型是给java编译器用的，编译后的binary不带泛型信息。
