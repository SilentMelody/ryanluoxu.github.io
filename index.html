乱七八槽

Posted on 2017-11-25

问题:

执行 hexo deploy --generate

得到： 

ERROR Deployer not found: github

解法：

in _configyml：

deploy:
    type: git   
    repository: https://github.com/Ryanluoxu/ryanluoxu.github.io.git  
    branch: master

in git bash:

npm install hexo-deployer-git --save
hexo g
hexo d

文件的三种状态

  • 已修改 modified ： 对某个文件进行修改，保存完成。
  • 已暂存 staged ： 执行 git add 之后
  • 已提交 committed ： 执行 git commit 之后

这里的提交目的地，是本地 repository。
常用的 github 为 remote repository。
如果需要将本地 repository 同步到 github 上，需要执行 git push。

用 Hexo 和 GitHub Pages 搭建博客

Posted on 2017-11-24

声明：本文为 ryanluoxu 原创文章，可以转载，但必须在明显位置注明出处！

搭建这个博客走了许多弯路。在这里分享总结之后的简化步骤。

1. 注册 github 账号

2. 登录后，添加一个 .github.io 结尾的 repository。复制地址。

3. 安装 Hexo。

3.1 安装 Git
3.2 安装 Node.js
3.3 安装 Hexo

4. 建立本地博客

4.1 本地建立一个博客文件夹。
4.2 打开 git bash，进入到该文件夹。
4.3 hexo init
4.4 git clone https://github.com/henryhuang/hexo-theme-aloha.git themes/aloha 
4.5 修改 _config.yml 里的 theme: aloha
4.6 hexo g
4.7 cd public/
4.8 git init
4.9 git add .
4.10 git commit -m "First Hexo Blog Post"
4.11 git remote add origin https://github.com/Ryanluoxu/ryanluoxu.github.io.git
4.12 git push origin master

5. 打开 https://ryanluoxu.github.io/
